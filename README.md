Unicorn is a complete, FIDE-rules-compliant chess engine

## Features

- Single file
- Board: pieces[144] (12x12)
- En passant
- Underpromotions (knight, bishop, rook)
- 2-fold repetition detection
- 50-move rule
- Stalemate and checkmate reported correctly to the GUI
